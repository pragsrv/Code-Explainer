📖 User Guide

🚀 Getting Started

Welcome to Code Explainer Pro! This tool helps you understand code by providing detailed explanations and syntax highlighting for multiple programming languages.

🎯 How to Use

  • Select Language: Choose your programming language from the buttons at the top
  • Enter Code: Type or paste your code in the text area
  • Get Explanations: Hover over highlighted keywords to see explanations
  • View Analytics: Check code statistics and complexity below

✨ Key Features

📝 Multi-Language Support

Supports JavaScript, Python, Java, C++, C#, HTML, CSS, and SQL with language-specific explanations.

🔍 Interactive Explanations

Hover over any keyword to see detailed explanations. Keywords are highlighted in red for easy identification.

🎨 Syntax Highlighting

Beautiful color-coded syntax highlighting makes your code easier to read and understand.

📊 Code Analytics

View detailed statistics including line count, keywords, functions, comments, and complexity analysis.

🛠️ Available Tools

  • 📝 Load Example: Load sample code for the selected language
  • 🎨 Format: Auto-format your code with proper indentation
  • 🗑️ Clear: Clear the code editor
  • 🌙 Dark Mode: Switch between light and dark themes

💡 Tips & Tricks

  • Use the Load Example button to see how different languages work
  • Hover over any red-highlighted keyword for instant explanations
  • The complexity indicator helps you understand code difficulty
  • Switch languages to see how similar concepts work across different programming languages
  • Use the format button to clean up messy code automatically

🎨 Theme Options

Switch between light and dark modes using the theme toggle button in the top-right corner. Your preference will be saved for future visits.
